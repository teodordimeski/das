Финалните исфилтрирани и целосно средени податоци се наоѓаат во binance_data_filter1_output.csv

1. crypto_downloader.py -> зема топ 1000 симболи(парови) и ги зачувува во binance_data_top1000.csv
2. filter1.py -> зема податоците од binance_data_top1000.csv ги филтрира и зачувува во binance_data_filter1_output.csv
3. filter2.py -> прави проверка за последниот датум на валутите, оние за кои има достапни податоци ги запишува во binance_data_filter1_output.csv заедно со датумот на последниот запис за нив, а оние за кои има само симбол без информации ги додава во missing.csv
4. filter3.py -> чута симболи за валутите за кои нема записи, ги превзема достапните податоци и ги додава во binance_data_filter1_output.csv без да додаде дупликат (напомена во binance_data_filter1_output.csv се наоѓаат веќе исфилтрираните податоци-сместени се податоци само за валидни валути)
